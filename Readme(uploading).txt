Process to upload webgl build on own website

. Zip file consisting Folders/Files -

	1. Build Folder ( build loader rar files )
	2. Template Folder ( consists logos and pngs for browser )
	3. index.html file
	
. Steps to upload on website -

	1. upload zip file on file manager of website.
	2. extract them 
	3. keep all the extracted files and folder in one place (in one folder)
	4. now, location of index.html file is our game link
	5. now you can play online with this link.
